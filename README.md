# snakemake

Running snakemake in GitHub actions.

## Inputs

### `directory`

**Required** The workdir for Snakemake. Default: `.`.

### `snakefile`

**Required** Snakefile containing the workflow description. Default: `Snakefile`.

### `snakefile`

**Required** Snakefile containing the workflow description. Default: `Snakefile`.

### `args`

**Required** Additional Snakemake arguments to use.
